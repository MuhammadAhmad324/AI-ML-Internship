## 📊 Task 1: Iris Dataset Analysis

### 🎯 Objective
To explore and visualize the Iris dataset using Python.

### 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

### 📂 Work Done
- Loaded Iris dataset using Seaborn
- Performed data exploration (shape, info, statistics)
- Visualized data using:
  - Scatter Plot
  - Histogram
  - Box Plot
  - Pair Plot

### 📈 Key Insights
- Dataset contains 150 samples and 3 flower species
- No missing values present
- Petal features are most important for classification
- Setosa is clearly separable from other species
